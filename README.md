# Guess the Animal! - 100 Fun Riddles for Kids

An interactive animal quiz game with 100 riddles for kids. Read three clues and guess which animal it is, then click to reveal the answer along with a picture from Wikipedia.

## Play Now

[https://ayeung.dev/animals-quiz/](https://ayeung.dev/animals-quiz/)

## Features

- 100 animal riddles with three clues each
- Click to reveal answers with fun emoji
- Animal photos fetched from Wikipedia
- Responsive card layout that works on desktop and mobile
- Single-page app with no dependencies
